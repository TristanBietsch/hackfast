# hackfast

A keyboard-first macOS distro.
Remove friction between thought and execution.

## What It Is

A modular guide for operating your machine with minimal interface friction.
Vimish navigation everywhere.
Universal keybinds across applications.
No mouse required.

## Quick Start

1. Check [prerequisites](docs/prerequisites.md)
2. Install [Homebrew](https://brew.sh)
3. Run the [bootstrap script](docs/scripts/bootstrap.sh.md)
4. Clone [dotfiles](https://github.com/TristanBietsch/dotfiles) and stow packages

## Tools

GUIs: Aerospace, Ghostty, Obsidian, Raycast, Shortcat, Karabiner Elements, Vimium

TUIs: nvim, tmux, ranger, lsd, gnu-stow, btop, opencode

## Philosophy

Keyboard-only means removing friction wherever possible.
Minimal hand movement.
Predictable, composable shortcuts.
Easy to learn incrementally.
