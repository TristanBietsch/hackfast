# Neovim Plugins

Plugins installed in the hackfast Neovim configuration.
Managed via lazy.nvim.

## Core

- lazy.nvim — plugin manager
- plenary.nvim — lua utility functions

## Completion

- blink.cmp — completion engine
- LuaSnip — snippet engine

## LSP

- nvim-lspconfig — LSP configuration
- mason.nvim — LSP installer
- mason-lspconfig.nvim — mason + lspconfig integration
- mason-tool-installer.nvim — auto-install tools
- lazydev.nvim — Neovim lua development
- fidget.nvim — LSP progress indicator

## Navigation

- telescope.nvim — fuzzy finder
- telescope-fzf-native.nvim — fzf algorithm for telescope
- telescope-ui-select.nvim — use telescope for vim.ui.select

## Editor

- nvim-treesitter — syntax highlighting and parsing
- guess-indent.nvim — automatic indent detection
- mini.nvim — collection of minimal modules
- todo-comments.nvim — highlight TODO comments
- which-key.nvim — keybind hints

## Git

- gitsigns.nvim — git decorations

## Theme

- tokyonight.nvim — colorscheme
